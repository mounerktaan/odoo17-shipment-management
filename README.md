# Odoo 17 Shipment Management Module

Custom Odoo 17 module developed to manage shipment workflows
for a freight brokerage platform.

## Features
- Shipment lifecycle management (Created, Picked Up, In Transit, Delivered)
- Integration with Odoo stock.picking
- Custom action buttons and views
- Role-based access control
- Clean and extendable module structure

## Tech Stack
- Odoo 17 Community
- Python
- XML (Views & QWeb)
- PostgreSQL

## Installation
1. Clone the repository
2. Copy the module into your custom_addons directory
3. Add the path to addons_path in odoo.conf
4. Restart Odoo
5. Install the module from Apps
